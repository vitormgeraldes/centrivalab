# CentrivaLab website

Static website for the CentrivaLab commercial platform.

## Positioning

CentrivaLab is the commercial platform built on the Orthogonal Centrifugal Membrane Filter (O-CMF) technology.

## Current product focus

- CentrivaLab 50/34 — reference configuration for 50 mL conical tubes and fixed-angle rotors around 34°
- CentrivaLab 94/25 — planned bench-top configuration for 94 mL rotor-class geometries around 25–26°

## Communication rules

- Use **CentrivaLab** as the brand name.
- Use **Orthogonal Centrifugal Membrane Filter (O-CMF)** as the scientific technology name.
- Keep engineering identifiers internal.
- Keep public technical claims aligned with validated data and documentation.
- Use RUO language where relevant.
